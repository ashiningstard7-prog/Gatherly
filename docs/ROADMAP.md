# Roadmap — Gatherly

## Phase 1 — MVP Foundation ✅
- Project setup, monorepo structure
- PostgreSQL schema (40+ models)
- REST API foundation
- Premium landing page
- Host / Farmer / Admin dashboards
- Seed data & documentation

## Phase 2 — Authentication (Next)
- Clerk integration (Google, GitHub, Apple)
- OTP login via SMS
- Email verification flow
- Onboarding wizard for new users

## Phase 3 — Payments
- Stripe checkout integration
- Razorpay for India market
- Wallet system
- Escrow payment flow
- Automatic refunds
- Host payout automation

## Phase 4 — Core Features
- Real-time chat (WebSocket)
- Google Maps integration
- Advanced search with geo-radius
- Calendar availability management
- QR ticket generation & scanning
- Push notifications

## Phase 5 — AI Features
- Recommendation engine
- Smart search (NLP)
- Price suggestions for hosts
- Review moderation
- Fraud detection ML model
- Chat assistant

## Phase 6 — Social
- Community feed
- Followers / following
- Stories
- Achievements & badges
- Leaderboards
- Referral program

## Phase 7 — Scale
- Mobile apps (React Native)
- Multi-language support
- Multi-currency
- Insurance integration
- Enterprise API
- White-label solution
