## Option C — Technical Deep-Dive (for engineer audience)

📐 Project deep-dive: **Gatherly** — experience marketplace architecture

I recently shipped an MVP structured for scale:

**Frontend:** Next.js App Router, server/client components, Tailwind CSS 4, Framer Motion, modular brand config

**Backend:** Express 5 middleware chain (Helmet → CORS → rate limit → RBAC → Zod validation), REST API versioning (`/api/v1`)

**Data:** 40+ Prisma models — users, experiences, bookings, payments, farm activities, social graph, audit logs. Hierarchical categories, indexed search fields.

**Infra:** npm workspaces monorepo, Docker Compose (Postgres + Redis), GitHub Actions build pipeline

**Security:** OWASP-aware — parameterized queries, input validation, role hierarchy, audit logging architecture

Not just a landing page — dashboards, seed data, API docs, deployment guides, and a Phase 2 roadmap (Clerk, Stripe, Maps, AI recommendations).

Repo: https://github.com/ashiningstard7-prog/Gatherly
